DOC sur les routes crées






PARTIE utilisateur






/auth/signup 
post > crée un utilisateur > données attendu
email, firstname, lastname, avatar, password.

/auth/login
post > login d'un utilisateur > données attendu
email, password

/auth/delete
delete > suppression d'un utilisateur > données attendu
email, password >>>> voir pour ne pas prendre le password en compte




/forum
get > Affiche les postes du forum

/forum/:id
get > Cherche l'article correspondant à l'id

/forum/post/:id
get > cherche les commentaires correspondant à l'id de l'article

/forum/create
post > Création d'un article > données attendu
profil_id, media, description, posted_date

/forum/update/:id
put > Modifier un article > données attendu
description profil_id

/forum/delete/:id
delete > Supprime un article > données attendu
profil_id

/forum/post/:id
get > Modifie un commentaire > données attendu
description, profil_id

/forum/post/:id
delete > Supprime un commentaire > données attendu
profil_id

/forum/post/
post > Publié un commentaire sur un poste > données attendu
profil_id, comment_text, post_id, comment_date
